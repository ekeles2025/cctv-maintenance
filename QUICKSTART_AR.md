🚀 **دليل البدء السريع - Quick Start Guide**

# البدء على Windows
=====================

1️⃣ **الطريقة الأولى - السهلة (موصى بها)**
   └─ افتح PowerShell من مجلد المشروع
   └─ اكتب: .\start.ps1
   └─ اضغط Enter
   └─ انتظر تثبيت المتطلبات
   └─ افتح المتصفح على: http://127.0.0.1:5000

2️⃣ **الطريقة الثانية - اليدوية**
   └─ افتح CMD أو PowerShell
   └─ انتقل لمجلد المشروع
   └─ اكتب الأوامر التالية:
   
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   python app.py
   
   └─ افتح المتصفح على: http://127.0.0.1:5000

# البدء على Linux/Mac
=======================

1️⃣ **الطريقة الأولى - السهلة (موصى بها)**
   └─ افتح Terminal من مجلد المشروع
   └─ اكتب: bash start.sh
   └─ اضغط Enter
   └─ انتظر تثبيت المتطلبات
   └─ افتح المتصفح على: http://127.0.0.1:5000

2️⃣ **الطريقة الثانية - اليدوية**
   └─ افتح Terminal
   └─ انتقل لمجلد المشروع
   └─ اكتب الأوامر التالية:
   
   python3 -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   python3 app.py
   
   └─ افتح المتصفح على: http://127.0.0.1:5000

# استكشاف الأخطاء الشائعة
==========================

❌ "Python is not installed"
   └─ قم بتحميل Python من: https://www.python.org
   └─ تأكد من إضافته لـ PATH

❌ "Module not found"
   └─ تأكد من تفعيل البيئة الافتراضية (venv)
   └─ ثم اكتب: pip install -r requirements.txt

❌ "Address already in use"
   └─ المنفذ 5000 مستخدم من قبل تطبيق آخر
   └─ غيّره في app.py: app.run(port=5001)

❌ "Database error"
   └─ احذف ملف camera_system.db وأعد التشغيل
   └─ سيتم إنشاء قاعدة بيانات جديدة تلقائياً
