# دليل استيراد المناطق من Excel

## زر "استيراد Excel" في صفحة المناطق

### الوصف
زر 📊 "استيراد Excel" يسمح باستيراد المناطق من ملف Excel.

### كيفية الاستخدام

1. **اذهب إلى صفحة المناطق**: `http://127.0.0.1:5000/regions`
2. **أو اذهب إلى مناطق سلسلة محددة**: `http://127.0.0.1:5000/chains/[chain_id]/regions`
3. **اضغط على زر** 📊 "استيراد Excel"
4. **اختر ملف Excel** بصيغة `.xlsx` أو `.xls`
5. **اضغط على زر** "استيراد"

### تخطيط الأعمدة في ملف Excel

| العمود | البيانات | مطلوب؟ |
|--------|----------|---------|
| A | **اسم المنطقة** | ✅ نعم |
| B | أي بيانات إضافية (اختياري) | لا |
| C | أي بيانات إضافية (اختياري) | لا |

### مثال على ملف Excel

```
| A         | B    | C    |
|-----------|------|------|
| القاهرة   |      |      |
| الإسكندرية |      |      |
| الجيزة   |      |      |
| المنصورة |      |      |
| طنطا    |      |      |
```

### ملاحظات هامة

- ✅ **الحقول الإلزامية**: اسم المنطقة (العمود A) فقط
- ✅ **مرونة الأعمدة**: يمكن وضع اسم المنطقة في أي عمود (سيتم استخدام أول عمود غير فارغ)
- ✅ **منع التكرار**: يتم تجاهل المناطق الموجودة بالفعل في نفس السلسلة
- ✅ **ربط بالسلاسل**: عند الاستيراد من صفحة سلسلة، يتم ربط المناطق تلقائياً بالسلسلة

### الرسائل المتوقعة

#### نجاح:
- `✅ تم إضافة X منطقة بنجاح من ملف Excel`
- `✅ تم إضافة X منطقة بنجاح لسلسلة 'اسم السلسلة' من ملف Excel`

#### تحذير:
- `⚠️ هناك X أخطاء: [تفاصيل الأخطاء]`
- `ℹ️ تم تجاهل X منطقة موجودة بالفعل`

#### خطأ:
- `❌ لم يتم اختيار ملف`
- `❌ يجب أن يكون الملف من نوع Excel (.xlsx أو .xls)`
- `❌ الملف فارغ أو لا يحتوي على بيانات صالحة`

### المسار التقني

- **Route**: `/regions/import-excel` أو `/chains/<int:chain_id>/regions/import-excel`
- **Method**: POST
- **Template**: `regions.html`
- **Function**: `import_regions_excel()`

### الأذونات

- فقط المدير (`admin`) يمكنه استخدام هذه الميزة

### استكشاف الأخطاء

1. **رسالة "اسم المنطقة إلزامي"**: تأكد من وجود اسم المنطقة في العمود A أو أي عمود غير فارغ
2. **ملف لا يعمل**: تأكد من أن الملف بصيغة `.xlsx` أو `.xls`
3. **لم يتم استيراد أي بيانات**: تحقق من وجود بيانات في الملف
4. **مناطق غير مستوردة**: قد تكون موجودة بالفعل، تحقق من رسائل التحذير
