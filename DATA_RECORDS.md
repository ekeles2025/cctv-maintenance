# سجل البيانات المسجلة في النظام

## 📊 أنواع البيانات المسجلة

### 1. الأعطال (Faults)
- **المسار**: قاعدة البيانات - جدول `fault`
- **البيانات**: 
  - اسم الكاميرا
  - نوع العطل
  - وصف العطل
  - تاريخ الإبلاغ
  - حالة العطل (مفتوح/مغلق)
  - اسم الفني
  - ملاحظات الإصلاح

### 2. الكاميرات (Cameras)
- **المسار**: قاعدة البيانات - جدول `camera`
- **البيانات**:
  - اسم الكاميرا
  - عنوان IP
  - الفرع التابع له
  - حالة الكاميرا

### 3. الفروع (Branches)
- **المسار**: قاعدة البيانات - جدول `branch`
- **البيانات**:
  - اسم الفرع
  - عنوان الفرع
  - أرقام الهواتف
  - المنطقة التابعة لها
  - نوع الفرع (دائم/موسمي)

### 4. المناطق (Regions)
- **المسار**: قاعدة البيانات - جدول `region`
- **البيانات**:
  - اسم المنطقة
  - السلسلة التابعة لها

### 5. السلاسل (Chains)
- **المسار**: قاعدة البيانات - جدول `chain`
- **البيانات**:
  - اسم السلسلة

### 6. المستخدمون (Users)
- **المسار**: قاعدة البيانات - جدول `user`
- **البيانات**:
  - اسم المستخدم
  - البريد الإلكتروني
  - الصلاحيات (admin/technician)

### 7. الملفات المرفوعة
- **المسار**: `static/uploads/`
- **البيانات**:
  - صور إصلاح الأعطال
  - مستندات مرفوعة
  - ملفات Excel المستوردة

## 📁 أماكن تخزين البيانات

### قاعدة البيانات الرئيسية
```
📁 camera_system.db
├── fault (الأعطال)
├── camera (الكاميرات)
├── branch (الفروع)
├── region (المناطق)
├── chain (السلاسل)
├── user (المستخدمين)
├── devicefault (أعطال الأجهزة)
└── bbmfault (أعطال BBM)
```

### الملفات الثابتة
```
📁 static/
├── uploads/ (صور الإصلاح والمستندات)
├── logo/ (شعار الشركة)
└── css/js/ (ملفات التصميم)
```

## 🔍 كيفية عرض البيانات المسجلة

### 1. عبر واجهة الويب
- **الأعطال**: `/faults`
- **الكاميرات**: `/cameras`
- **الفروع**: `/branches`
- **المناطق**: `/regions`
- **السلاسل**: `/chains`

### 2. عبر قاعدة البيانات مباشرة
```sql
-- عرض الأعطال المسجلة
SELECT * FROM fault ORDER BY date_reported DESC;

-- عرض الكاميرات المسجلة
SELECT * FROM camera ORDER BY name;

-- عرض الفروع المسجلة
SELECT * FROM branch ORDER BY name;
```

## 💾 كيفية الحفاظ على البيانات

### النسخ الاحتياطي المنتظم
```bash
# نسخ قاعدة البيانات (يحتوي على كل البيانات)
copy camera_system.db backup\camera_system_YYYYMMDD.db

# نسخ الملفات المرفوعة
xcopy static\uploads backup\uploads_YYYYMMDD /E /I
```

### استيراد/تصدير البيانات
- **تقارير Excel**: `/reports/excel`
- **تصدير الأعطال**: زر التصدير في صفحة الأعطال
- **استيراد الكاميرات**: زر استيراد Excel في صفحة الكاميرات

## 📈 إحصائيات مهمة

### البيانات التي يتم تتبعها:
- عدد الأعطال لكل فرع
- عدد الكاميرات لكل فرع
- الأعطال حسب النوع
- أداء الفنيين
- الفروع الأكثر عرضة للأعطال

## 🔄 تحديث البيانات

### البيانات التي تتغير تلقائياً:
- حالة الأعطال (عند الإصلاح)
- تواريخ الإبلاغ والإصلاح
- حالة الكاميرات
- تعيينات الفنيين

### البيانات التي تحتاج تحديث يدوي:
- معلومات الفروع (هواتف، عناوين)
- صور الإصلاح
- ملاحظات الفنيين

---
**💡 ملاحظة**: كل البيانات محفوظة في قاعدة البيانات `camera_system.db` - هذا هو الملف الرئيسي الذي يجب الحفاظ عليه!
