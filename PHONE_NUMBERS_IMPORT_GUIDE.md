# استيراد أرقام التليفونات من Excel

## الوصف
تم إضافة ميزة جديدة لاستيراد أرقام التليفونات من ملف Excel للفروع الموجودة في النظام.

## كيفية الاستخدام

### 1. الذهاب إلى صفحة الفروع
- افتح صفحة الفروع لأي منطقة: `http://127.0.0.1:5000/branches/[region_id]`

### 2. الضغط على زر استيراد أرقام التليفونات
- ستجد زر 📱 "استيراد أرقام التليفونات" بجانب زر استيراد Excel العادي

### 3. تحميل ملف Excel
- يجب أن يكون الملف بصيغة `.xlsx` أو `.xls`
- **العمود C**: أسماء الفروع (يجب أن تطابق أسماء الفروع في النظام تماماً)
- **العمود D**: أرقام التليفونات

### 4. مثال على ملف Excel

| العمود A | العمود B | العمود C | العمود D |
|----------|----------|----------|----------|
|          |          | فرع 1    | 01234567890 |
|          |          | فرع 2    | 01123456789 |
|          |          | فرع 3    | 01098765432 |

## ملاحظات هامة

- ✅ يتم تحديث رقم التليفون الأول فقط (`phone_number`)
- ✅ يعمل فقط على الفروع الموجودة في نفس المنطقة
- ⚠️ أسماء الفروع يجب أن تطابق تماماً الأسماء في النظام
- ⚠️ إذا لم يتم العثور على فرع، سيظهر تحذير بأسماء الفروع غير الموجودة
- 📊 سيتم إظهار عدد الفروع التي تم تحديثها بنجاح

## الرسائل المتوقعة

### نجاح:
- `✅ تم تحديث أرقام التليفونات لـ X فرع بنجاح`

### تحذير:
- `⚠️ فروع غير موجودة: فرع 1, فرع 2`

### خطأ:
- `❌ خطأ في قراءة الملف: [رسالة الخطأ]`

## المسار التقني

- **Route**: `/branches/import-phone-numbers/<int:region_id>`
- **Method**: POST
- **Template**: `branches.html`
- **Function**: `import_phone_numbers_excel()`
