# 📹 نظام إدارة الكاميرات - Camera Management System

## نظرة عامة
نظام ويب شامل لإدارة شبكات الكاميرات الأمنية، تتبع الأعطال، وإدارة الفنيين والمستخدمين.

## المتطلبات
- Python 3.8+
- pip (مدير الحزم)
- SQLite3 (مُثبت افتراضياً مع Python)

## المميزات الرئيسية ✨
- ✅ إدارة السلاسل والمناطق والفروع والكاميرات
- ✅ نظام تتبع الأعطال الشامل
- ✅ إدارة الفنيين وتعيين الأعطال
- ✅ تقارير Excel قابلة للتنزيل
- ✅ نظام مصادقة آمن
- ✅ واجهة رسومية سهلة الاستخدام بـ العربية
- ✅ معالجة تواريخ ومناطق زمنية صحيحة

## التثبيت والبدء السريع

### 1. استنساخ المشروع
```bash
git clone <repository-url>
cd "Last Project pro"
```

### 2. على Windows (الطريقة الموصى بها)
```powershell
# تشغيل سكريبت البدء
.\start.ps1
```

### 3. على Linux/Mac
```bash
# تشغيل سكريبت البدء
bash start.sh
```

### 4. بدء يدوي
```bash
# إنشاء البيئة الافتراضية
python -m venv venv

# تفعيل البيئة
# على Windows:
venv\Scripts\activate
# على Linux/Mac:
source venv/bin/activate

# تثبيت المتطلبات
pip install -r requirements.txt

# تشغيل التطبيق
python app.py
```

## التكوين

### إعدادات البيئة (.env)
```env
# مفتاح سري آمن
SECRET_KEY=your-secret-key-here

# قاعدة البيانات
DATABASE_URL=sqlite:///camera_system.db

# إعدادات التطبيق
COMPANY_NAME=اسم شركتك
LOGO_SIZE=50
MAX_CONTENT_LENGTH=16777216
```

### الملفات المهمة
- `app.py` - تطبيق Flask الرئيسي
- `config.py` - إعدادات التطبيق
- `utils.py` - دوال مساعدة
- `requirements.txt` - متطلبات Python

## استكشاف الأخطاء

### المشكلة: خطأ في الدخول
- تحقق من اسم المستخدم وكلمة المرور
- تأكد من أن قاعدة البيانات موجودة

### المشكلة: الصور لا تُرفع
- تحقق من الإذن على مجلد `static/uploads/`
- تأكد من أن صيغة الملف مدعومة (PNG, JPG, GIF, WebP, SVG)

### المشكلة: خطأ في قاعدة البيانات
- حذف ملف `camera_system.db` وأعد التشغيل
- تحقق من توفر مساحة كافية على القرص
