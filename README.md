# 🎥 Camera Management System

نظام إدارة كاميرات المراقبة والأعطال - بسيط وموثوق.

## 🚀 التشغيل السريع

### 1. تثبيت الحزم المطلوبة
```bash
# استخدم بيئة افتراضية مفضلة (موصى به)
python -m venv .venv
.
# ثم شغّل الـ venv (CMD) أو فعّله في PowerShell
.venv\Scripts\activate.bat
# أو في PowerShell: & .\.venv\Scripts\Activate.ps1

pip install -r requirements.txt
```

### 2. التشغيل
```bash
# انقر مرتين على run.bat
run.bat

# أو تشغيل مباشر
python app.py
```

### 3. الوصول
- **التطبيق**: http://localhost:5000
- **المدير**: admin / admin123
- **الفني**: فني1 / tech123

## 📋 الملفات المهمة

| الملف | الاستخدام |
|-------|-----------|
| `app.py` | الكود الرئيسي |
| `run.bat` | تشغيل سريع |
| `requirements.txt` | الحزم المطلوبة |
| `camera_system.db` | قاعدة البيانات |
