# عرض سريع للبيانات المسجلة - كيفية فتحها وإغلاقها

## 🚀 تشغيل المشروع وعرض البيانات

### 1. تشغيل المشروع
```bash
# في مجلد المشروع
python app.py

# أو
python run_app.py
```

### 2. فتح المتصفح
- **الرابط المحلي**: http://127.0.0.1:5000
- **الرابط على Railway**: https://respectful-empathy-production-c573.up.railway.app

## 📊 أماكن البيانات المسجلة

### الصفحة الرئيسية للبيانات
```
http://127.0.0.1:5000/dashboard
```

### عرض كل نوع من البيانات:

#### 🔧 الأعطال المسجلة
```
http://127.0.0.1:5000/faults
```
- كل الأعطال التي تم إبلاغها
- حالة كل عطل (مفتوح/مغلق)
- اسم الكاميرا والفرع
- تاريخ الإبلاغ والإصلاح

#### 📷 الكاميرات المسجلة
```
http://127.0.0.1:5000/cameras
```
- كل الكاميرات في النظام
- عنوان IP لكل كاميرا
- الفرع التابع له

#### 🏪 الفروع المسجلة
```
http://127.0.0.1:5000/branches
```
- كل الفروع المسجلة
- عناوين وأرقام الهواتف
- المنطقة التابعة لها

#### 🗺️ المناطق المسجلة
```
http://127.0.0.1:5000/regions
```
- كل المناطق في النظام
- السلسلة التابعة لها

#### 🔗 السلاسل المسجلة
```
http://127.0.0.1:5000/chains
```
- كل السلاسل التجارية

## 💾 حفظ وإغلاق المشروع

### 1. أخذ نسخة احتياطية قبل الإغلاق
```bash
# نسخ قاعدة البيانات (يحتوي على كل البيانات)
copy camera_system.db backup\camera_system_backup.db

# نسخ الملفات المرفوعة
xcopy static\uploads backup\uploads_backup /E /I
```

### 2. إغلاق المشروع
- في نافذة Terminal: اضغط `Ctrl + C`
- أو أغلق نافذة Terminal مباشرة

## 🔄 فتح المشروع مرة أخرى

### 1. تشغيل المشروع
```bash
cd "d:\التعديل\CCTV Camera"
python app.py
```

### 2. فتح المتصفح على نفس الروابط
- كل البيانات ستكون كما تركتها
- الأعطال والكاميرات والفروع كلها محفوظة

## 📋 قائمة سريعة بالبيانات المهمة

### البيانات التي ستجدها دائماً:
- ✅ **الأعطال**: كل عطل تم إبلاغه
- ✅ **الكاميرات**: كل كاميرا مسجلة
- ✅ **الفروع**: كل فرع في النظام
- ✅ **المناطق**: كل منطقة مسجلة
- ✅ **المستخدمون**: كل حساب في النظام
- ✅ **الصور**: كل صور الإصلاح المرفوعة

### البيانات التي لا تضيع أبداً:
- كلها محفوظة في `camera_system.db`
- عند إغلاق المشروع وإعادة فتحه، كل شيء كما هو

## 🎯 أهم 3 روابط للحفظ في المفضلة

1. **لوحة التحكم**: http://127.0.0.1:5000/dashboard
2. **عرض الأعطال**: http://127.0.0.1:5000/faults
3. **تقرير Excel**: http://127.0.0.1:5000/reports/excel
